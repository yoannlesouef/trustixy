# Roadmap

## Phase 1 — Core Product (Weeks 1–4)

Week 1:
* Setup project
* Auth (user + organization)
* DB schema (multi-tenant: organizations, users, ai_systems)

Week 2:
* CRUD AI systems (scoped to organization)
* Dashboard UI

Week 3:
* Risk classification (LLM)
* Compliance document generation (LLM)

Week 4:
* Export (PDF / text)
* Polish UI
* Landing page with dual CTA (Start free / Become a partner)

## Phase 2 — Partner Channel (Weeks 5–8)

Week 5:
* Partner application flow (form + tier selection)
* Partner data model (partners, partner_clients)
* Partner API endpoints

Week 6:
* Partner portal UI (client list, invite client)
* Branded onboarding email for invited clients

Week 7:
* White-label settings (logo, colors, custom domain)
* Commission tracking dashboard

Week 8:
* Partner portal polish
* End-to-end partner → client flow testing

## Phase 3 — Scale (Weeks 9+)

* Marketplace listings (AWS, Azure, Google Cloud)
* API for embedded integrations
* Advanced audit features
* Multi-user workspaces within organizations
* PDF export enhancements
* Improved classification accuracy
